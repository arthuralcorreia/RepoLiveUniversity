# RepoLiveUniversity

Repositório referente ao processo seletivo da Live University

## Processo de inicialização

Para dar inicio a a plicação, clone o repositorio, entre no repositorio referente ao mesmo e digite os seguintes comandos no terminal.

``` bash

cd SpecificationforFrontMiddleEndTestNoA2
npm install
DEBUG=specificationforfrontmiddlendtestnoa2:* nodemon start

```
